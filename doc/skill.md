# 任务界面
## 显示
按照进行中、已过期、已完成的顺序显示任务，分别显示在不同的分类中。每个标题栏设置不同的颜色
- 进行中：绿色
- 已过期：红色
- 已完成：灰色
任务显示样式：
- 任务名称：加粗显示
- 任务截止时间：显示在任务名称下方
- 是否为周期任务：设置标签进行显示
- 所属分类：设置标签进行显示
- 任务优先级：根据优先级显示不同的颜色
  - 重要且紧急：红色
  - 重要但不紧急：橙色
  - 不重要但紧急：黄色
  - 不重要且不紧急：绿色
可以切换到不同的任务分类中，分别显示不同的任务。
当在不同分类中切换时，顶部的导航栏会显示当前所在的分类。
不同分类显示的样式同任务列表界面。
在不同分类中若无进行中的任务、已过期的任务时，显示提示信息-"所有任务已完成"
## 任务详情
点击任务时，会跳转到任务详情界面，显示任务的详细信息。可以对任务进行编辑、删除操作。
显示信息包括：
- 任务名称，在任务名称前面有一个复选框，用于标记任务是否已完成，可以点击复选框来切换任务的完成状态。
- 任务优先级：显示任务优先级，用下拉框选择，包括重要且紧急、重要但不紧急、不重要且紧急、不重要但不紧急四个选项。
- 任务分类：显示任务分类，用下拉框选择，包括无、用户设置的任务分类等选项。
- 任务截止时间：显示任务的截止时间，用日期选择器选择，包括年、月、日、时、分等。
- 任务周期: 
    - 如果任务有周期，会显示任务的周期：
        - 任务周期为每周重复时，会显示重复的具体日期，如每周一、每周三等。
        - 如果任务周期为每月重复时，会显示重复的具体日期，如每月1日、每月15日等。
    - 如果没有周期，可以选择是否设置周期：
        - 可以选择任务的周期，如每周重复、每月重复等。
        - 若选择周期为每周重复，还可以选择重复的具体日期，如每周一、每周三等；如果选择周期为每月重复，还可以选择重复的具体日期，如每月1日、每月15日等。
- 任务描述
- 设置提醒: 可以选择是否设置提醒，以及提醒时间。
在最下方会显示三个按钮：
- 取消：点击后会返回上一个界面。
- 删除任务：点击后会删除该任务。
- 保存任务：点击后会保存任务的修改。
当删除任务时，会弹出确认删除的弹窗，确认后会删除该任务。
当删除任务、保存任务后，先显示操作是否成功，成功后再返回上一个界面。

# 日历界面
在最顶部显示当前年、月，能够切换到上一个月、下一个月，以及切换到下一年、上一年。
点击日期可以查看该日期的任务列表。
显示点击日期中的周期任务在之后的日期中的任务个数
任务列表的显示效果参考任务界面的显示效果

# 清单界面
显示该用户个人清单、共享清单
按照个人清单、共享清单分类显示清单。显示当前分类下的清单个数、可以折叠当前分类的清单
每个清单显示样式：
- 清单名称：加粗显示
- 清单描述：显示在清单名称下方
- 清单中的任务数量：显示在清单名称下方，用括号括起来
